# Food Venue Selector App

 Food Venue Selector App Using React + Redux + Webpack + React Router

Libraries:
- React + Redux + Webpack + React Router

**To Run locally:**
```
$ npm install
$ npm run start
```
Try http://localhost:8080

**To Run test:**
```
$ npm run test
$ npm run test:watch
```

**To build for production:**
```
$ npm install
$ npm run build
```
